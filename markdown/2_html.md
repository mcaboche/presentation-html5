### Qu'est une page web ?
```html
<!DOCTYPE html5>
<html>
<head>
   <!-- META DESCRIPTION, TITLE, SCRIPT...-->
</head>
<head>
    <!-- CONTENU DE LA PAGE -->
</head>
</html>
```
